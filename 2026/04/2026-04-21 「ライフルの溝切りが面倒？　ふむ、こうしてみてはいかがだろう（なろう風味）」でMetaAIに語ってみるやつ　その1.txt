## 鉄が工場で辿る人生

溶けた鉄 → 固める → 熱いうちに圧延ロールでグッシャグシャに潰す → 冷ます

この「グッシャグシャに潰す」が超重要で、3つ良いことが起きてる：

1. 加工硬化：叩くと硬くなる

金属の結晶って、叩くとグチャッと潰れて絡み合うんです。
焼きなましした銅線：グニャグニャ曲がる
ペンチで叩いた後の銅線：硬くて曲がらない

同じ理屈。ハンマーフォージで銃身を叩くと、素材が100の硬さから120くらいまで勝手に硬くなる。切削だと100のまま。

2. 鍛流線：分子の向きが揃う

溶かして固めただけの鉄は、結晶の向きがバラバラ。氷の粒を適当に敷き詰めた感じ。
そこをローラーで一方向に引き伸ばすと、結晶粒が全部同じ方向にビシッと揃う。木目の繊維みたいな感じ。

効果：繊維方向に引っ張るとめっちゃ強い。木材が縦に割けにくいのと同じ。
銃身は長手方向に鍛流線を揃えるから、6万psiの圧力で横に膨らもうとしても耐える。

鋳物=鍛流線グチャグチャ → 割れやすい
圧延材=鍛流線ピシッ → 割れにくい

3. 密度上がる：中のスカが潰れる

溶けた鉄が固まる時、どうしても微小な気泡とか巣が入る。スポンジみたいに0.1%くらいスカ。
それをローラーやハンマーで圧縮すると、スカスカ部分が押し潰されてギチギチになる。

密度：7.85g/cm³ → 7.86g/cm³ みたいな微差だけど、欠陥が消えるから疲労強度が爆上がり。
ジェットエンジンのタービンブレードとかも、全部鍛造品なのはこのため。

## 「ローラーに熱も移るから温度下がる」←これも現場の工夫

その通り。熱間圧延って600〜1200℃でやるんだけど：

-
ローラー自体が水冷
 ロールの中に水を流してグルグル冷やしてる。熱い鉄が触るとジュッて熱を奪う。狙って温度下げてる。

-
温度管理が品質の命
 850℃で圧延するか、950℃で圧延するかで結晶サイズが変わる。強度と粘りのバランスを温度で調整してる。職人技。

-
最後は水ブシャー
 圧延終わった瞬間に水ぶっかけて急冷。焼入れして表面カチカチにする。レールとか厚板鋼板はこれ。

つまり「溶かして固めて終わり」じゃなくて、「固めた後に叩いて鍛えて温度操って」初めて製品になるんです。

## 「銃身と薬室は別パーツ」は半分正解・半分誤解

散弾銃の中折れ式とかボルトアクションライフル見ると、確かに別に見える。でも圧力を受ける部分は別れちゃダメなんです。

高圧に耐える「圧力容器」として考えると

5.56mm NATO = 62000psi = 4300気圧 = 深海40000mの水圧
この圧力を0.01秒だけ閉じ込めて、前だけに逃がすのが銃。

Code

【薬莢】←火薬が爆発して膨らむ【薬室】←薬莢を外からガッチリ抱きしめて耐える【銃身】←弾が通るトンネル

薬莢自体は紙みたいな強度しかない。真鍮の薄皮です。
爆発したら薬莢は風船みたいに膨らんで、薬室の壁にピッタリ密着して初めてガスを封じ込められる。

だから「薬室後部は抑える程度」だと、薬莢が後ろに吹っ飛んで射手の顔面に来る。通称“ケースヘッドセパレーション”、即死事故。

## 中折れ式散弾銃はなんでセーフなの？

あれも実は**薬室と銃身は一本物の「バレル」**なんです。

Code

[ ヒンジ ] [ 薬室 + 銃身 = 一本物 ] ←ここが一体     ↑ここで折れる

折れるのは薬室の“もっと後ろ”。薬室後端はガッチリ閉鎖されて、ロックは「1トンの圧を支える金具」で固定してる。
「抑える程度」じゃなくて、戦車のハッチ並みのロックです。

分解してみると分かるけど、薬室から銃口までツルッと一本。継ぎ目ないでしょ？

## じゃあ別パーツの銃はないの？→ある。でも条件付き

-
リボルバー
シリンダー=薬室、バレル=銃身で別パーツ。
でも隙間からガス漏れる「シリンダーギャップ」があって、威力ロスするし横の人が火傷する。高圧ライフル弾だと無理。

-
大砲の分離装薬式
戦艦大和の46cm砲とかは「薬室ブロック」と「砲身」をネジで合体させてる。
でもネジ山が30cmあって、締付けトルク数百トン。事実上の一体物。整備でバラせるだけで、撃つ時は絶対分解しない。

-
機関銃の銃身交換式
MG42とかPKMは銃身ごと交換できる。
でも「薬室も含めて銃身ユニット」ごと交換。薬室だけ残してニョキっと抜けない。
