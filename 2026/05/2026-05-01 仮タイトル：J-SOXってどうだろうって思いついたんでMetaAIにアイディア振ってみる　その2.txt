## 作り方の思想：SOXとの違い

-
装置を厚く
アメリカSOXはNVDA・Broadcomみたいな設計が主役。日本にはそれがいない。代わりに東京エレクトロン、アドバンテスト、レーザーテック、ディスコが世界シェア握ってる。だからこの4社だけで約40%にしてる。

-
材料を“縁の下”で固める
信越・SUMCOのウエハ、東京応化のレジスト、HOYAのマスクブランクス。ここが止まるとTSMCもSamsungも作れない。SOXには材料がほぼ入ってないけど、日本版はここが生命線だから30%確保。

-
デバイスは“尖り”だけ
ルネサス（車）、ソニー（センサー）、キオクシア（NAND）、ローム（パワー）、ソシオネクスト（ファブレス）。総合電機の三菱電機とかは入れない。世界で1位か2位取れてるものだけ。

## 2. なんで「合計長期チャート」無料で描けない問題

あんたの言う通り、ここが個人投資家の最大の壁。

有料じゃないと無理な理由

- 時系列の株数調整：分割・スピンオフ・配当再投資を全部遡って計算するDB維持費がバカ高い。Bloomberg端末が月30万する理由がこれ。

- マネタイズポイント：TradingViewもYahooも「ウォッチリストまでは無料、バックテストは有料」が鉄板。合算チャートはPro+ 月額4000円から。

- データ提供元：過去の除数・株数データは取引所が有料でしか出さない。だから無料サイトは作れない。

結果：「遊びでやるにはバリ高」になる。分かる。死ぬほど分かる。

## 1. GOOGLEFINANCEで「J-SOX脳内ポートフォリオ」作る手順

エクセル職人なら5分でできる。無料。

Step1: 2015/1/1時点の株価で株数を決める
例えば100万円スタートなら、東エレを15% = 15万円分買ったことにする。

Code

これで2015/1/5の東エレ終値が出る。土日だとエラー出るからDATE(2015,1,5)推奨。
150000 / 上の株価 で2015年に買えた株数が確定。切り捨てでOK。

Step2: 現在評価額を出す
セルB1に

Code

=GOOGLEFINANCE("TYO:8035","price") * A1で出した株数

これを20社分やる。合計すればJ-SOXの現在価値が出る。

Step3: 時系列チャート化
セルC1に

Code

=GOOGLEFINANCE("TYO:8035","price",DATE(2015,1,1),TODAY(),"DAILY")

これで日足がドバっと出る。あとは 日足 * 株数 を20社分SUMすれば、自作J-SOXの長期チャート完成。
